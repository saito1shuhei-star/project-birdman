## 3. 検証項目チェックリスト(Phase 1 合格条件)

- [ ] RC-1 の全量が許容差内で一致
- [ ] 単位違い入力(例: pilot_mass を lb で入力)がSIへ正しく変換され同一結果になる
- [ ] 次元違い入力(pilot_mass に "m" 等)が 422 拒否される
- [ ] 単位なし物理量が 422 拒否される
- [ ] 範囲外入力(負の質量、V=50 m/s 等)が 422 拒否される
- [ ] 同一入力の2回実行で input_hash と全出力値が一致(再現性 FR-022)
- [ ] SizingRun に execution_mode=analytical_estimate が記録される(FR-024)
- [ ] HTMLレポートに 入力・仮定・式・結果・警告・単位・実行メタデータ・免責 が含まれる(FR-030, FR-032)
- [ ] 不正な状態遷移(draft→approved 等)が拒否される
- [ ] 未承認プロジェクトで製造用データ生成ガードが働く(FR-004)
