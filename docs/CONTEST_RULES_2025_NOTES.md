## 2. 発進・プラットホーム(規約1条・2条・巻末図)

| 項目 | 値 | 出典 |
|---|---|---|
| 発進方式 | 原則としてプラットホームから**自力による助走**で水上へ発進(カタパルト等の外部補助なし) | 1条a |
| 機体保持者 | 発進時**3名まで**(発進時にプラットホームから落下してはならない) | 1条b, d |
| 飛行準備の制限時間 | プラットホーム上で**5分** | 1条e |
| プラットホーム高さ | **10m**(水面より) | 2条a |
| 助走路の長さ | **10m**(パイロットは先端から10m以内の地点から助走開始) | 2条b |
| 助走路の傾斜 | **3.5度**(下り) | 2条c |
| プラットホーム概寸 | 発進方向 約20m(8.7+7.3+4.0m)× 幅 約12m。**寸法は概寸・予告なく変更あり(予定図)** | 巻末図 |
| 手すり | 片側 +750mm、他側 +900mm | 巻末図 |
| その他 | 大会旗ポール(プラットホーム上から7m)、クレーンあり、助走路はウレタン敷 | 巻末図 |

**設計上の含意**:
- 離陸解析(Phase 2–3)の初期条件: 助走10m(3.5度下り)+高さ10mの降下で得られる速度が初期飛行速度を決める(ASSUMPTIONS A-115)
- プラットホーム幅約12mに対し翼幅20〜35mの機体は**翼端が手すり(+750/+900mm)の上を通過する**。
  翼取付高さ+上反角で、片側約6m位置の翼下面高さが手すりを確実に上回る設計とすること(Phase 2平面形設計での確認項目)

## 3. 飛行に関する規定(規約3条)

| 項目 | 内容 | 出典 |
|---|---|---|
| 飛行禁止区域 | 1. プラットホーム湖側先端から湖岸側の水面(=後方飛行禁止) 2. 陸地・島 3. 漁場・浅瀬・岩礁等レスキュー不能区域、沖ノ島水道等 4. 橋梁・桟橋・構造物周辺 5. 全船舶周辺 6. **湖岸より350m以内の水域**(プラットホーム周辺の指定飛行区域を除く) | 3条a |
| 飛行制限高度 | **プラットホームの高さ(水面より10m)を超えてはならない** | **3条b** |
| 危険回避 | 禁止区域接近・高度超過時はパイロット自らの判断で回避。指示があれば即時従う | 3条c |
| ナビゲーター | **人力プロペラ機部門限定**: チームスタッフ1名が大会の機体伴走ボートに乗船し、飛行監視・進路変更・高度低下・着水指示を行う | 3条d |
| 罰則 | 区域1進入・区域2〜6接近進入は失格。制限高度超過は回避しない場合等に失格 | 3条e |

> ⚠️ **ルールブック内部の不整合(発見事項)**: 3条e罰則2項は「飛行禁止区域**7**」への接近・進入に言及するが、
> 3条aの列挙は**1〜6のみ**であり区域7は定義されていない。前年版の名残と推定されるが、
> 出場時に大会事務局へ確認すること(チーム側の作業)。

## 4. 機体に関する規定(規約4条)— 設計制約の中核

前提: **「プラットホームから安全に発進できる機体に限る」**(4条柱書)。

| 項目 | 内容 | PBMへの反映 |
|---|---|---|
| サイズ・重量・形状 | **制限なし**(a, b, c) | 反映済み(A-101: 翼幅制限はチーム独自制約) |
| 緊急離脱構造 | 着水・緊急時にパイロットが**ただちに機体より離脱できる構造**であること(d1) | CON-009。Phase 3コックピット設計の必須確認項目 |
| 安全設計 | 危害を及ぼす恐れのある形状・材料・装備品の禁止。**あらゆる着水態勢を想定しパイロットを衝撃から守る安全設計・製作**(d2) | CON-009。人間承認事項 |
| 自作限定 | 自作人力飛行機に限る(e柱書) | 前提 |
| 使用禁止装置 | 浮昇補助(カイト・気球など)禁止。**内外部蓄積エネルギーなどパイロット以外の動力禁止**(e1) | CON-006/007。PBMの推進計算はパイロット出力のみを前提(現行実装は適合) |
| 使用禁止付属品 | 発進後の**部品落下禁止**(車輪投下等)。プラットホーム上の**レール使用禁止**。**遠隔操縦機器禁止**(e2) | CON-008。離陸解析で車輪落下方式を仮定しない |

## 5. 飛行距離の測定(規約5条)

- 滑空機部門: 完全着水後、**機体の中で最もプラットホームに近い部分**から、水面上に投影したプラットホーム先端までの直線距離を測定
  - 設計上の含意: 距離は機体後端(最もPFに近い部分)基準。着水姿勢・機体分解のしかたも記録に影響し得る
- 人力プロペラ機部門の測定ルールは**別紙**に定める(**別紙は未入手** — チームで入手すること)
- 1,000m未満: 三角測量 / 1,000m以上: GPS(5条a, b)

## 6. 人力プロペラ機部門の競技ルール(規約6条)

| 項目 | 値 |
|---|---|
| コース | 南ルート往復(PF⇔沖島ポイント)+北ルート往復(PF⇔竹生島ポイント)で**合計70km**。どちらを先に飛ぶかはパイロット判断(a, b) |
| 距離 | PF⇔沖島=PF⇔竹生島=**直線18km**。各ポイント⇔プラットホームゾーンは**17km**(d) |
| プラットホームゾーン | PF先端から湖側**半径1km**の円周内(d) |
| 旋回 | 各ポイントクリア時・ルート折返し時は**パイロン旋回必須**(c) |
| 完全飛行 | 両ルート往復後、**着水ゾーン**(PF先端から湖側半径**300m**の円周内)に着水、または着水ゾーン到達後に禁止区域外の湖面に着水(e) |
| 同着時の順位 | **複数チームが70km達成時は飛行時間の短いチームが上位**(g) |

**設計上の含意**:
- 70km完走を狙う設計は**パイロン旋回(計6回以上)の旋回性能・旋回時荷重**が必須の設計ケースになる(ASSUMPTIONS安全率メモ参照)
- 6条gにより、トップ争いでは**巡航速度が順位を決める** → Phase 4最適化の評価関数に「所要時間最小化」を含める根拠
- 現行のCALCULATION_SPEC(定常水平飛行)は巡航区間の近似のみ。旋回・往復戦略はSimulation機能(Phase 5+)で扱う

## 7. パイロットに関する規定(規約7条)

- 泳げること(b)、ヘルメット着用(c)、必要に応じライフジャケット(d)
- ハーネス等でパイロットと機体を接続・固定する場合(主に滑空機)は**クイックリリース装置**(当日事務局で用意)を着用(e)
- 年齢: **出場申込時の満年齢18歳以上**。大会当日20歳未満は保護者同意書(f)
- フライト前後のメディカルチェック受診(g)、パイロット自身も機体安全チェックを行う(h)
